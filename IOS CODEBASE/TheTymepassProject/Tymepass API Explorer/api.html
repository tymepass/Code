      • POST https://tyme-pass.appspot.com/newEvent

        • Creates a new event

        Implementation Notes

        Creates a new event from the parameters passed and returns the new event's (server) Id

        Parameters

        Parameter Value Description
        title title of the event
        info info of the event
        stealth if the creator will appear as stealth from the event
        attending attending value for the creator-event relationship
        isAllDay Is it an all day event?
        location location of the event
        creatorId id of the user that creates the event
        message array of messages that the creator sent on creation of the event
        recurring Is it recurring?
        startTime startTime of the event
        endTime endTime of the event
        dateModified dateModified of the event
        dateCreated dateCreated of the event
        isPrivate is the new event Private?
        isOpen is the new event Open?
        isGold is the new event gold?
        isEditable is the new event editable by the invitees?
        iCalId the new event's ical Id (for the creator- event relationship)
        photo event photo (not handled yet)
        Hide Response

        Response

        Server ID of new event

         
        1. [
        2.     {
        3.         "id":353008
        4.     }
        5. ]
